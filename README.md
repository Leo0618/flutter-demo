### 官方flutter命令使用国内镜像配置，Windows添加环境变量如下：
    
	键: PUB_HOSTED_URL  
	值：https://pub.flutter-io.cn

	键: FLUTTER_STORAGE_BASE_URL  
	值：https://storage.flutter-io.cn
